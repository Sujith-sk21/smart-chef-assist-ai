Smart Chef Assist AI (Fridge Hero)

Smart Chef Assist AI is a simple web application that helps users create recipes using just three ingredients. By leveraging the Groq API (LLaMA 3 model), the app generates a creative dish name along with an easy-to-follow 3-step recipe. The goal is to make cooking quick, fun, and accessible, especially when you have limited ingredients at home.

Features

Allows users to input any three ingredients
Generates a unique dish name based on the inputs
Provides a clear and simple 3-step recipe
Includes a clean and modern user interface with smooth animations

Tech Stack

HTML, CSS, JavaScript
Groq API (LLaMA 3 large language model)

How to Run

Open the index.html file in a browser
Enter any three ingredients
Paste your Groq API key when prompted
Click on "Generate Dish" to get the recipe

Note

The API key is entered manually by the user and is not stored anywhere, ensuring basic security.
